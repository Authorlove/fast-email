# 投标邮件极速发送系统 - 使用文档

## 系统简介

投标邮件极速发送系统是一款专为投标场景设计的邮件发送工具，支持毫秒级精准定时发送、多SMTP服务器管理、附件上传和发送日志记录等功能。

## 系统架构

- **前端**: React + TypeScript + Vite + Tailwind CSS
- **后端**: Express + Node.js + Nodemailer
- **通信**: RESTful API

## 快速启动

### 方式一：使用 Setup 脚本自动初始化（推荐）

适用于首次使用或新环境部署，脚本会自动检测系统环境并安装所需依赖。

```bash
# 赋予执行权限
chmod +x setup.sh

# 运行初始化脚本
./setup.sh
```

脚本会自动完成以下操作：
- 检测操作系统（macOS / Linux）
- 安装/更新 Node.js（要求 v18+）
- 安装项目依赖
- 创建启动脚本（`start.sh` 和 `start.bat`）

初始化完成后，使用以下命令启动系统：

```bash
# 同时启动前后端服务
./start.sh
```

### 方式二：手动启动

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动后端服务

```bash
npm run server
```
后端服务将在 http://localhost:3001 启动

#### 3. 启动前端服务

```bash
npm run dev
```
前端服务将在 http://localhost:5173 启动

#### 4. 访问系统

打开浏览器访问 http://localhost:5173

## 功能模块

### 1. 撰写邮件

#### 填写邮件内容
- **收件人邮箱**: 输入目标邮箱地址
- **邮件主题**: 输入邮件标题
  - 点击主题栏右侧的盾牌图标可添加防重放UUID
- **邮件正文**: 输入邮件内容

#### 附件上传
- 点击"点击上传附件"按钮选择文件
- 支持多文件上传
- 单文件大小限制：10MB
- 显示文件Hash值用于完整性校验

#### 定时发送
- **目标发送时间**: 设置精确到毫秒的发送时间
- **提前启动时间**: 设置提前启动的毫秒数（建议100ms，用于补偿网络延迟）
- 点击"设置定时发送"启用定时任务

#### 立即发送
- 点击"立即发送"按钮立即发送邮件
- 显示发送状态和服务器响应时间

### 2. SMTP配置

#### 添加SMTP服务器
1. 点击"添加服务器"按钮
2. 填写服务器信息：
   - 服务器名称：自定义名称
   - 服务器地址：SMTP服务器地址
   - 端口：587（STARTTLS）或465（SSL）
   - 用户名：邮箱地址
   - 密码/授权码：邮箱授权码
   - SSL/TLS：根据端口选择

#### 测试连接
- 点击服务器卡片上的信号图标测试连接延迟
- 或点击"测试所有服务器"批量测试

#### 启用/禁用服务器
- 点击播放/暂停图标切换服务器状态

### 3. 发送日志

#### 查看日志
- 显示所有邮件发送记录
- 包含时间、状态、响应码、服务器响应时间等信息

#### 导出日志
- 点击"导出CSV"按钮导出日志为CSV文件

#### 清空日志
- 点击"清空日志"按钮清除所有记录

### 4. 系统设置

#### NTP时间同步
- 显示当前同步状态和时间偏移
- 点击"立即同步"手动同步时间
- 自动每小时同步一次

#### 安全设置
- 自动添加防重放UUID
- 附件Hash校验
- SMTP密码加密存储

## 常见问题

### Q1: 后端服务器离线怎么办？
**A**: 确保已运行 `npm run server` 启动后端服务，并检查3001端口是否被占用。

### Q2: 邮件发送失败提示SSL错误？
**A**: 
- 587端口：使用STARTTLS，secure设为false
- 465端口：使用SSL，secure设为true
- 已自动处理，无需手动配置

### Q3: 附件上传失败？
**A**: 
- 检查文件大小是否超过10MB限制
- 检查浏览器控制台是否有错误信息
- 系统使用SHA-256计算文件Hash

### Q4: QQ邮箱授权码如何获取？
**A**:
1. 登录QQ邮箱网页版
2. 设置 → 账户 → 开启SMTP服务
3. 按提示发送短信验证
4. 获得16位授权码

### Q5: 定时发送不准确？
**A**:
- 调整"提前启动时间"参数（建议100-200ms）
- 确保NTP时间同步正常
- 考虑网络延迟因素

## 技术参数

### 性能指标
- 发送延迟：≤50ms（含网络传输）
- 时间精度：≤10ms
- 并发能力：单线程发送间隔≤10ms

### 支持协议
- SMTP (端口25/465/587)
- SSL/TLS加密
- STARTTLS

### 浏览器支持
- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

## 安全说明

1. **密码安全**: SMTP密码使用AES-256加密存储
2. **防重放攻击**: 支持在邮件主题添加UUID
3. **附件完整性**: 发送前校验文件Hash
4. **传输安全**: 支持SSL/TLS加密传输
